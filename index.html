  • Chrono Power Action

    A productivity launcher for Chrome and Edge, which can search everything in your browser, including opened tabs, bookmarks, history, downloads, short URLs, quick commands and more. You can also set up short URL mappings and search URL templates to navigate to or search your favorite websites by typing a few letters. There are many commands allow you type less to achieve more, such as:
    • built-in calculator
    • full keyboard navigation support
    • mute/unmute tabs
    • sort - Sort tabs in current window
    • dedupe - Close duplicated tabs in all windows
    • urls - Copy tab URLs in current window
    • rands - Generate a random string
